# Hardware_Test_Tool

本项目是为了在没有充足硬件设施的环境下提供工具，保证测试的顺利进行。目前已搜罗的工具已经包括：`lsblk`，`lsscsi`，`lspci`和`megacli`。

详细介绍请阅读：[硬件环境测试环境模拟][1]

## 下载代码

```
git clone git@github.com:tony-yin/Hardware_Test_Tool.git
```

## 安装

```
cd Hardware_Test_Tool
./build install
```

## 卸载

```
cd Hardware_Test_Tool
./build uninstall
```
